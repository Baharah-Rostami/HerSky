# American Greatness

A simple front-end project built with **HTML** and **Tailwind CSS** for practice purposes. This project explores notable U.S. presidents and highlights their contributions to American history.

## About the Project

American Greatness is a responsive website that presents information about the four presidents whose faces are carved into Mount Rushmore:

* George Washington
* Thomas Jefferson
* Theodore Roosevelt
* Abraham Lincoln

The website also includes a separate section featuring quotes, comments, and notable statements from more recent U.S. presidents such as:

* Joe Biden
* Donald Trump
* Barack Obama

## Features

* Responsive design using Tailwind CSS
* Clean and modern user interface
* Information about Mount Rushmore presidents
* Dedicated page for modern presidential quotes and comments
* Mobile-friendly layout

* Tailwind CSS
